# React Boilerplate

React JS boilerplate which contains `react-router-dom`, `redux`, `jest`, `enzyme`, `eslint`, `husky` configured init.

## Run Project

`npm start`

OR

`yarn start`

## Build Project

`npm run build`

OR

`yarn build`

## Serve Build

`npm i -g serve`

`npm run serve`

OR

`yarn serve`

## Test Project

`npm run test`

OR

`yarn test`
